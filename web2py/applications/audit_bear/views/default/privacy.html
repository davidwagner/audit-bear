Privacy

Results produced from the iVotronic logs are temporarily stored on our server and served based on session cookies. Accessing this site over http means that an attacker could potentially replay the session cookie and view a user's audit results. The uploaded files are deleted before the results request is completed.

If any of this is a concern, it would be simple to create a local version of this website using our Google Code repository. We would be happy to assist anyone interested in using our code.
